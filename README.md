# **Portfolio Website**

## **Description**

This is my personal portfolio website showcasing my skills as a backend developer with a focus on building modern web applications using Django and Node.js, along with React for the frontend. It demonstrates my projects, blogs, and technical expertise, with a responsive design, dark mode toggle, and smooth navigation.

The website highlights my **backend expertise**, my **projects**, and my passion for solving real-world problems through technology. It’s designed to be a professional showcase for recruiters, clients, and collaborators.

---

## **Tech Stack**

**Frontend:**

* React.js
* Tailwind CSS

**Backend:**

* Node.js (for contact form email functionality)
 

**Other Tools:**

* Git & GitHub
* VSCode
---

## **Features**

* Responsive design for mobile and desktop
* Dark mode toggle
* Smooth scroll navigation to sections: About, Skills, Projects, Blogs, Contact
* Centralized mobile menu (hamburger)
* Projects section with backend API integration
* Blog section for sharing technical insights
* Contact form: Sends messages to email Via Node.js backend
